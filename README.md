# TriWave-2.1

A little 2.1 channel audio system from scratch. i got dedicated subwoofer plus left and right stereo output. Used classd amps so it stay cool and efficent and doesnt get too much power. desgined in easyeda.

# WHATS INSIDE?

- TPA3118D2
- HUSB238
- 3.5mm Audio Jack
- 100nF
- 10Ω
- 10kΩ
- SUB, LEFT, RIGHT speakers
- USB-C
- 2-layer board [80x55mm]

# POWER STUFF
| Power source        | Voltage | Amp Draws |
|-----------------|-----------|----------|
| USB-C        | 12V       | 2A|
| Direct 12V pad        | 12V     |  2A|
| Bass      | 12V      |  5A |
